# Image Bank

Pacote inicial de imagens SVG para o SONIC SEARCH.

## Estrutura

- `covers/`: capas por estilo musical (usadas no app)
- `social/`: cards 1200x630 para compartilhar com amigos
- `logos/`: logo marca e logo horizontal
- `manifest.json`: indice com metadados para integrar no frontend

## Uso rapido

- Exibir uma capa no app: ler `manifest.json` e montar caminho relativo
- Gerar previews para redes: usar arquivos em `social/`
- Avatar/app icon inicial: `logos/neonpulse-logo-mark.svg`

## Proximo passo recomendado

Converter os SVG para WebP (1x e 2x) e manter SVG como master para edicao.
