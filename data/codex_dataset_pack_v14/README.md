# Codex dataset pack

## Conteúdo
- data/psytrance_db_v14.jsonl (891 artistas)
- data/psycore_db_v14.jsonl (psycore auditado)
- AGENTS.md (instruções que o Codex lê automaticamente)
- CODEX_PROMPT.txt (prompt pronto)

## Como usar (2 passos)
1) Copie a pasta `data/` e o arquivo `AGENTS.md` para a raiz do seu projeto.
2) Abra o Codex nesse projeto e cole o conteúdo de `CODEX_PROMPT.txt`.

Obs: O Codex lê AGENTS.md automaticamente antes de começar a trabalhar.
